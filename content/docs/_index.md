## Twemoji Versions

Twemojis are retrieved directly from the [Twemoji GitHub repository](https://github.com/twitter/twemoji) based on a particular [release version](https://github.com/twitter/twemoji/releases).

Versions supported:

- `14.0.2`
- `14.0.1`
- `14.0.0`
- `13.0.1`
- `13.0.0`
